# **🛒 Carrinho de Compras JS**

## __🛍️ Sistema de Carrinho de Compras Interativo__

    Este projeto consiste em uma aplicação web que simula um carrinho de compras moderno, permitindo ao usuário adicionar, remover e visualizar produtos de forma dinâmica e intuitiva.

    Desenvolvido com foco em boas práticas de Front-end, o sistema proporciona uma experiência semelhante a de e-commerces reais, trabalhando conceitos essenciais para a construção de interfaces interativas.

    
__🎯 Objetivo__

    Consolidar conhecimentos em desenvolvimento Front-end, com foco em interatividade, organização de código e construção de interfaces funcionais.


**_💻 Sobre o Projeto_**

    O Carrinho JS foi criado como parte do processo de aprendizado no curso Técnico em Desenvolvimento de Sistemas, com o objetivo de aplicar na prática conhecimentos fundamentais de desenvolvimento web.

**_A aplicação permite:_**

Adicionar produtos ao carrinho
Remover itens
Atualizar quantidades
Visualizar o valor total da compra em tempo real
- ⚙️ Funcionalidades
- 🧩 Manipulação dinâmica do DOM com JavaScript
- 🖱️ Uso de eventos (click) para interação do usuário
- 📱 Layout responsivo (Mobile First)
- 🧮 Cálculo automático de valores totais
- 🔢 Controle de quantidade de produtos
- 🎨 Estilização moderna com CSS (Flexbox)
- 🧱 Estrutura organizada com HTML semântico


### __🛠 Tecnologias Utilizadas__
- HTML5
- CSS3
- JavaScript
- Git
